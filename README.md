## 1. Roles
- **Admin**
  - Manage users (Consumers, Contractors)
  - Approve/monitor jobs
  - Handle disputes
  - System settings & analytics

- **Consumer**
  - Post jobs
  - Manage dashboard (Active, In Progress, Completed jobs)
  - View & accept/reject contractor quotes
  - Communicate with contractors
  - Rate & review completed jobs
  - Manage account & notifications

- **Contractor**
  - Browse available jobs
  - Submit quotes (bid price, description, portfolio images)
  - Communicate with consumers
  - Update job progress
  - Build reputation via ratings & reviews
